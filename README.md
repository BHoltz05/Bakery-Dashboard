# Bakery Operations Analytics Dashboard

## Project Overview

This project is an operational analytics solution designed to simulate real-world bakery performance reporting. It demonstrates how raw operational data can be transformed into structured business insights using Excel and Power BI.

The system focuses on analysing sales performance, operational waste, staffing efficiency, and weekly trends to support business decision-making.

---

## Tools and Technologies

- Microsoft Excel (data preparation, KPI calculations, pivot tables)
- Power BI (interactive dashboard development)
- Data modelling and aggregation techniques
- Business reporting and operational analysis

---

## Project Structure

### Excel Workbook
- Raw operational dataset (Logbook)
- KPI calculations and summaries
- Product performance analysis
- Waste analysis
- Staffing efficiency analysis

### Power BI Dashboard
- Executive summary dashboard
- Product performance analysis
- Waste analysis
- Staffing and weekly performance analysis
- Interactive filters (slicers)

---

## Key Performance Indicators (KPIs)

- Total Revenue
- Units Sold
- Waste Units
- Waste Value
- Wastage Rate
- Revenue per Staff Hour

---

## Key Insights

- Revenue is concentrated in a small number of high-performing products.
- Waste is disproportionately generated by specific product categories.
- Staffing efficiency varies across different weeks of operation.
- Weekly revenue trends highlight fluctuations in operational performance.

---

## Dashboard Features

- Interactive Power BI visuals
- Product-level performance breakdown
- Waste and cost impact analysis
- Weekly operational trend analysis
- Filtering by product category and week
